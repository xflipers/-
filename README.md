# Красный Кремль

Интерактивный сайт об истории Красной площади: временная линия, памятники и здания, исторические личности, даты и кликабельная галерея.

## Что внутри

- `index.html` - структура страницы.
- `style.css` - оформление в красных, тёмно-зелёных, кремовых и золотых цветах.
- `script.js` - интерактивные карточки, временная линия и модальные окна.

Сайт статический, его можно открыть напрямую через `index.html` или опубликовать через GitHub Pages.
